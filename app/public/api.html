Vote API

A microservice to host and vote on polls.

New Poll

Create a new poll

Route

https://vote-jp.herokuapp.com/api/new

POST

Body format:

{
"title":"What's your favorite flavor?
"choices":"Chocolate, Vanilla"
}

Response:

{
"status": "200",
"id": "H1DV9P35-",
"title": "What's your favorite flavor?",
"choices": [
"Chocolate",
"Vanilla"
]
}

View Poll

Route

https://vote-jp.herokuapp.com/api/<poll id>

GET
